# Simulador de Grúa de un Barco de Carga en Realidad Virtual usando Unity

Simulador de realidad virtual desarrollado en Unity que recrea las operaciones de una grúa de carga en un entorno portuario. Permite controlar la grúa, mover contenedores y completar misiones mediante una experiencia inmersiva orientada al aprendizaje y entrenamiento de operaciones portuarias.

---

## Características

- Simulación de operaciones portuarias.
- Control de la grúa en un entorno de realidad virtual.
- Manipulación de contenedores.
- Sistema de misiones.
- Sistema de puntuación.
- Dificultad dinámica mediante balanceo del barco.
- Audio ambiental y efectos sonoros.
- Interfaz intuitiva para el usuario.

---

## Tecnologías utilizadas

- Unity
- C#
- XR Interaction Toolkit
- OpenXR
- Blender
- Git y GitHub

---

## Requisitos

- Unity Hub.
- Unity 2022 LTS o superior.
- Visual Studio 2022.
- Meta Quest 2/3 o dispositivo VR compatible (opcional para ejecución en realidad virtual).

## Setup

1. Clonar el repositorio:

```bash
git clone https://github.com/kitto-hue/-Simulador-de-Gr-a-de-un-Barco-de-Carga-en-Realidad-Virtual-usando-Unity.git
```

2. Abrir el proyecto desde Unity Hub.

3. Esperar a que Unity importe todos los recursos y paquetes necesarios.

4. Verificar que estén instalados los paquetes XR necesarios.

5. Ejecutar la escena principal desde el Editor de Unity o generar la compilación para el visor de realidad virtual.

---

## Uso

1. Ejecutar la aplicación.

2. Seleccionar el modo de simulación.

3. Utilizar los controles de realidad virtual para operar la grúa.

4. Transportar los contenedores hacia la ubicación indicada.

5. Completar las misiones para obtener la mayor puntuación posible.

## Estructura del proyecto

```
Assets/
│── Scenes/        # Escenas del simulador

```

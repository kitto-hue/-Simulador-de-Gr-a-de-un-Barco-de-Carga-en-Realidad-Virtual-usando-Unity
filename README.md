# -Simulador-de-Gr-a-de-un-Barco-de-Carga-en-Realidad-Virtual-usando-Unity
Simulador de realidad virtual desarrollado en Unity que recrea las operaciones de una grúa de carga en un entorno portuario. Permite controlar la grúa, mover contenedores y completar misiones mediante una experiencia inmersiva orientada al aprendizaje y entrenamiento de operaciones portuarias.
